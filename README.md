# OhSoft OCam

## Overview

OhSoft OCam is a screen recording and screenshot capture tool designed for Windows users. It provides a straightforward way to record selected areas of the screen or capture still images, supporting various video and image formats. The software aims to facilitate screen recording tasks with clear and accessible controls suitable for different user needs.

## Key Features

**Selective screen recording with adjustable capture area**  
**Support for multiple video formats including AVI and MP4**  
**Screenshot capture with image format options such as PNG and BMP**  
**Real-time audio recording alongside video capture**  
**Simple user interface for quick access to recording functions**  
**Customizable frame rates and video quality settings**  
**Pause and resume recording capabilities**  
**Option to save recordings to user-defined locations**
